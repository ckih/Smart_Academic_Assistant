# Security Policy

## Reporting a Vulnerability

The Smart Academic Assistant team takes security seriously. If you discover a security vulnerability in this project, please report it privately through one of the following methods:

1.  **Email**: Please send an email to [security@example.com](mailto:security@example.com) with a description of the vulnerability and steps to reproduce it.
2.  **GitHub Private Reporting**: If available for this repository, you may use the GitHub security advisory feature to report vulnerabilities.

### Our Process

Upon receiving a vulnerability report, we will:

- Acknowledge the receipt of the report within 48 hours.
- Investigate the issue and provide a timeline for a fix.
- Notify the reporter once the vulnerability is addressed.

Thank you for helping us keep the Smart Academic Assistant secure!
